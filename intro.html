1. 플랫폼 개요

VET EX는 법정화폐(P2P) 거래의 복잡한 과정에 대한 문제를 해결하기 위한 탈중앙 거래 플랫폼입니다. USDT 및 HEX 토큰 거래에서 판매자와 구매자 사이의 중개 리스크를 스마트컨트랙트가 대신 보증하며, 거래 수수료는 생태계 토큰(VET)의 가치 상승과 보상 구조로 연결됩니다.

핵심 특징
- 중앙 운영자 없이 스마트컨트랙트로 거래 보호
- USDT / HEX 기반 에스크로 거래 지원
- HEX 거래 시 구매자에게 VET 보너스 자동 지급
- 거래 수수료는 vetBank에 누적되며 주 1회 VET 보유자에게 환원
- 오프체인 정보(SNS, 계좌)는 Firebase로 관리하여 UX 최적화
핵심 목표
- 중앙 제거 + 신뢰 강화
- 수수료 수익을 참여자에게 환류
- 오프체인 UX 최적화와 온체인 비용 절감

2. 해결하려는 문제

기존 P2P 코인·법정화폐 거래의 구조적 문제는 다음과 같습니다.

문제
- 중앙거래소 가입 절차의 어려움
- 입금 후 잠수, 중복 입금, 허위 클레임
- 중앙 중개자의 임의 개입 위험
- 수수료 수익이 운영사에 집중
VET EX의 해결
- 토큰은 컨트랙트에 에스크로 예치
- 상태: OPEN → TAKEN → PAID → RELEASED로 명확히 관리
- 구매자·판매자·중재자 권한 분리
- 수수료는 VET 생태계로 환류

3. VET EX의 해결 방식

VET EX는 중앙을 제거하되 신뢰를 강화하는 구조를 목표로 설계되었습니다. 핵심은 자금/거래/정보 레이어의 분리입니다.

레이어 분리
1) 자금 신뢰 레이어
2) 거래 행위 레이어
3) 정보·UX 레이어
원칙
- 운영자는 자금 이동 권한 없음
- 온체인은 자금과 결과 기록
- 오프체인은 개인정보·메시지·평판 처리
3-1. 스마트컨트랙트 기반 에스크로
- 구매자가 USDT 또는 HEX를 에스크로 컨트랙트에 예치
- 판매자는 오프체인으로 실물 또는 법정화폐 전달
- 구매자 확인 시 스마트컨트랙트가 자동 정산
- 분쟁 발생 시, 명시된 조건에 따라 환불 또는 정산 실행

중앙 운영자가 자금을 보관하지 않으며, 운영자는 자금 이동 권한을 갖지 않습니다.
3-2. 오프체인 정보 분리 구조
- 계좌 정보, SNS, 메신저 ID
- 거래 상태 메시지
- 사용자 평판 및 히스토리

오프체인 정보는 지갑 주소를 키값으로 Firebase에 관리되며, 블록체인에는 거래 결과/금액/해시만 기록됩니다.

이를 통해 개인정보 노출 최소화, 빠른 UX, 온체인 비용 절감을 동시에 달성합니다.

4. 거래 흐름 (User Flow)

1. 판매자
거래 조건 등록 (금액, 받고 싶은 통화, 거래 수단, 한도)
2. 구매자
거래 선택. 판매 목적의 USDT/HEX는 컨트랙트에 예치되어 안전
3. 오프체인 결제
은행 이체 또는 대면거래 가능. 법정화폐/실물 거래 진행
4. 거래 확정
구매자 확인 → 자동 정산. 수수료 처리. HEX 거래 시 VET 보상
4-2. 분쟁 발생 시
- 일정 시간 내 확인 미이행 시
- 컨트랙트 조건에 따라 자동 환불 또는 관리자 개입 트리거

관리자는 판단자가 아니라 조건 실행을 보조하는 역할만 수행합니다.

5. 토큰 구조 및 경제 모델

5-1. VET 토큰의 역할
VET는 단순 수수료 토큰이 아니라 플랫폼 수익 참여 권리를 의미합니다.

- 거래 보너스 지급
- 수수료 환원 대상
- DAO 의사결정 참여
- 장기 보유 인센티브
5-2. 거래 수수료 구조
- 모든 거래에는 소정의 수수료 발생
- 수수료는 즉시 vetBank로 적립
- 운영비 차감 후 잔여 수익 누적

누적 수익은 매주 1회 VET 보유자에게 비례 분배됩니다.
5-3. HEX 거래 보너스 구조
- HEX 거래 시 구매자는 거래 즉시 VET 보너스 지급
- 지급된 VET는 락업 또는 즉시 유통 선택 가능
- DAO 수익을 실제 거래 참여자에게 재분배

참여 기반 수익 분배 구조로 시장 활성화를 촉진합니다.

6. vetBank 구조

vetBank는 VET EX의 핵심 수익 허브입니다.

기능
- 모든 거래 수수료 누적
- 수익 분배 기준 계산
- 장기 수익 풀 형성
수익 분배 기준(개념)
(잔고 / 스테이킹한 VET) × (1 / 기본값 500)

운영 정책은 DAO 투표로 조정 가능합니다.
특징
- 단일 관리자 출금 불가
- DAO 규칙에 따른 분배만 가능
- 온체인 잔고 투명 공개

vetBank는 커뮤니티 공동 금고입니다.

7. DAO 거버넌스

VET EX는 단계적으로 DAO로 전환됩니다.

DAO에서 결정되는 항목
- 수수료 비율 조정
- 보너스 지급 정책
- 분쟁 처리 기준
- 신규 통화·토큰 추가
- 운영 예산 사용

VET 보유량에 따라 투표 가중치가 적용됩니다.

8. 보안 및 리스크 관리

8-1. 스마트컨트랙트 보안
- 자금 이동 로직 단순화
- 관리자 권한 최소화
- 재진입 공격 방지 적용
8-2. 사용자 리스크 관리
- 거래 한도 설정
- 신규 계정 제한
- 평판 기반 거래 노출

1단계

10. 결론

VET EX는 실물·법정화폐 거래를 탈중앙 금융 구조로 흡수하는 플랫폼입니다. 중앙의 신뢰를 제거하고 수익을 참여자 모두에게 환원하며, 거래 그 자체가 DAO의 성장 동력이 되는 구조를 지향합니다.

거래를 하면 할수록 플랫폼의 주인이 되는 시장을 목표로 합니다.
